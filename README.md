# Top One Percentile — Static Website

Static website for **Top One Percentile (TOP)** coaching institute, built with pure HTML, CSS, and vanilla JavaScript. Hosted via GitHub Pages.

## Pages
- `index.html` — Home page
- `programs.html` — All coaching programs (Board, JEE, NEET, NTSE, KVPY)
- `faculty.html` — Faculty profile
- `reviews.html` — Student testimonials
- `gallery.html` — Gallery
- `faq.html` — Frequently Asked Questions
- `contact.html` — Contact & Enquiry form (WhatsApp integration)

## Hosting
This site is deployed via **GitHub Pages** from the `main` branch root.
